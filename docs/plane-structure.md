# Plane.so 워크스페이스 구조 설계

> Daily Quests 시스템 스펙(`start.spec.md`) + 개인 로드맵(`myplan.md`) 기반

---

## 1. 워크스페이스 개요

```
Workspace: "Pulse"
├── [P1] 일본 IT 취업 마스터플랜        ← 중장기 프로젝트 (6개월)
├── [P2] 영어권 IT 취업 마스터플랜      ← 중장기 프로젝트 (1년)
├── [P3] AWS 자격증                     ← 중장기 프로젝트 (단기)
├── [P4] SCP 유튜브                     ← 사이드 프로젝트
├── [P5] 건강 관리                      ← 중장기 프로젝트 (지속)
└── [루틴] Daily Quests 일일 퀘스트       ← 루틴 프로젝트 (고정 1개)
```

---

### [루틴] Daily Quests 일일 퀘스트

> Daily Quests 시스템이 매일 자동 생성하는 퀘스트 전용 프로젝트

#### States (이슈 상태)

| State | Group | 설명 |
|---|---|---|
| To-Do | unstarted | 오늘 해야 할 퀘스트 |
| In Progress | started | 현재 진행 중 |
| Deferred | backlog | 미완료로 연기됨 |
| Done | completed | 완료 |
| Canceled | cancelled | 취소됨 |

#### Labels (원본 프로젝트 추적용)

| Label | 색상 제안 | 용도 |
|---|---|---|
| `P1-일본취업` | 🔴 Red | P1에서 복사된 루틴 |
| `P2-영어취업` | 🔵 Blue | P2에서 복사된 루틴 |
| `P3-AWS` | 🟠 Orange | P3에서 복사된 루틴 |
| `P4-SCP유튜브` | 🟣 Purple | P4에서 복사된 루틴 |
| `P5-건강관리` | 💚 Teal | P5에서 복사된 루틴 |
| `기타루틴` | ⚪ Gray | 프로젝트 미연동 루틴 |
| `llm-generated` | 🟢 Green | LLM이 추가 제안한 퀘스트 |
| `user-created` | 🟡 Yellow | 유저가 수동 추가한 퀘스트 |
